Mission

Welcome to the tracking website for the St. John's Streets Project. In November 2019 after seeing a post by Reddit user /u/davisvilums I was inspired to attempt a similar project in my home town. I currently track my rides using the Strava Android application. I then export the .GPX files from the Strava webpage and combine them using a free program - GPS Babble. An online tool GPS visulaizer is used to map all the GPX line simultaniously allowing me to keep track of what streets I have completed.

Future Plans
